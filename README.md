# ESS — Employee Self Service App

A mobile application that allows employees to conveniently access work-related services such as attendance logs, payslips, personal profile, and request submissions.  
This app is strictly **ESS-only** and does **not** include HR/admin modules.

---

## Features

### Authentication
- Account Setup (for newly hired employees)
- Login
- Forgot Password & OTP verification
- Reset Password

### Dashboard
- Quick summary of attendance, requests, and payslip availability
- Shortcut navigation to major modules

### Attendance
- Monthly attendance summary
- Daily logs
- QR code for time-in/time-out (if supported by backend)

### Payslip
- List of payslip periods
- Detailed breakdown of earnings and deductions

### Profile
- View personal and employment information
- Update editable fields (address, contact details, etc.)

### Requests
- Unified requests for Leave, Overtime, and General Requests
- Track status and view request details

### Notifications
- Alerts for approved requests, new payslips, and system updates

### Settings
- Change password
- Logout
